Advanced Space Wars! Play Here!



Description

This HTML5 game was developed for the COMP397 course (Web Game Programming). This is space shooter game based on the box2d engine where players take control of a ship and use it too destroy other enemies!

Some features of this Game include

  • Realistic Physics!: game based on Box2d engine
  • Lasers: PEW PEW!
  • Torpedoes: POW POW!
  • Debris: Destroy your enemies and watch their ships parts scatter across space
