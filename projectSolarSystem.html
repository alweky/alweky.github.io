ThreeJS Solar System Play Here!



Description

This ThreeJS model that was developed for the COMP392 course (Advacned Graphics). This is a solar system model which includes all eight planets (distances not to scale) and two moons of Saturn.

Some features of this Game include

  • Speed Scroll: increase or decrease how fast the planets are moving
  • Stop Simulation: you can also pause as well
