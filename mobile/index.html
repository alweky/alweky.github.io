Description

This app was developed for the COMP304 course (Wireless Programming). It allowed users to enter in vital data such as heart-rate, blood pressure, etc... and send these inputs to an web service.

Some features of this app include

  • User Login: login screen can take both patient and nurse login
  • Vital Sign: allow the input of vital signs by a nurse that will be sent to an external database
  • Watch Videos: motivational video to promote patients to exercise more
  • Signs and Symptoms Checklist: check certain symptoms and receive possible illnesses
  • Send and Receive Health Text: allows Nurses and Patients to send and receive texts of health related issues
  • Location Tracking: allows Nurses too locate Patients
