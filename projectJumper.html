Jumper! Play Here!



Description

This HTML5 game was developed for the COMP397 course (Web Game Programming). This is platforming game based on the box2d engine where players take control of a gum stick and have it transverse the many lands that present itself to it

Some features of this Game include

  • Realistic Physics!: game based on Box2d engine
  • Water Level: not only can you jump around in this game, but in the second world, you can also swim
